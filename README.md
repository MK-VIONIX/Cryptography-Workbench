# Cryptopals Crypto Challenges — Solutions

Implementing and breaking cryptographic systems from scratch.

This repository contains my solutions to the **Cryptopals Crypto Challenges**, a well-known set of exercises designed to teach **practical cryptography through implementation and attacks**.

Instead of relying heavily on external libraries, the solutions focus on **manual implementation of cryptographic primitives and attacks** to understand the mathematics and weaknesses behind real cryptographic systems.

---

## What are Cryptopals Challenges?

The Cryptopals challenges are a collection of progressively difficult problems that teach practical cryptography and cryptanalysis.

They start with simple encoding problems and gradually move toward real cryptographic attacks such as:

* XOR cipher attacks
* Breaking repeating-key encryption
* Implementing AES
* ECB detection
* CBC bit-flipping attacks
* Padding oracle attacks
* Hash length extension attacks
* RSA attacks

These challenges are widely recommended for anyone who wants to **truly understand how cryptography works internally**.

---

## Goals of This Repository

* Learn **cryptographic primitives**
* Understand **how encryption schemes fail**
* Implement attacks used in **real cryptanalysis**
* Strengthen **low-level programming and algorithmic thinking**
